Some of my Work

Meet-App

This App gets a list of fictitious upcoming Development events around the world. It is a Progressive Web Application (PWA) written with a Test Driven Development (TDD) approach. The data comes from a private Google Calendar (using Serverless functions to fetch the Google Calendar API). It also implements an OAuth2 Authorization server, for user to login with a Google account. The UI was built with React and includes chart visualizations that respond to changing events.

TopFlix Angular

A Single Page Application (SPA) about movies. Users can register/login to view a list of recommended movies and display their details like: Genres, Directors, Synopsis and select their favorites ones. Built with Angular, TypeScript & Angular Material. Together with the myFlix backend API, conforms a MEAN fullstack App.

TopFlix React

A client-side Single Page Application (SPA) that showcases a list of recommended movies. Users can: register their own account to login; create their own list of favorite movies that persists in server, and view details about movie plots, genres and Directors. Built with React, Redux and Bootstrap, TopFlix consumes the myFlix API conforming a MERN fullstack application.

myFlix REST API

A Public Web Server that provides several endpoints to query information from handpicked movies. Built from the ground up with Node.js, Express and MongoDB, it implements authentication & authorization using JWT tokens. myFlix serves as the backend for both: Topflix-React & TopFlix-Angular applications.

Pokedex

Pokedex is a simple application that fetches all Pokemon characters from a public API. Built with HTML, CSS, JavaScript, jQuery and styled using Bootstrap, users can search and display vector images and details from their favorite characters.
