Some of my Work

Meet-App

This App gets a list of upcoming Development events around the world. It is a PWA written with a TDD approach, it consumes a Google Calendar API with Serverless functions and an OAuth2 Authorization server to login with a Google account. The UI was built with React, and includes chart visualizations that respond to changing events.

TopFlix Angular

A Single Page Movies Application. Users can register and login to view Movies, Genres, Directors, Synopsis & like their favorites ones. Built with Angular, TypeScript & Angular Material. Together with the myFlix backend API, conforms a MEAN fullstack Application.

TopFlix React

A Single Page App for movies that consumes myFlix API. Users can register and login to view Movies, Genres, Directors & choose their favorites. Built with React, Redux and Bootstrap. The fullstack conforms a MERN Application.

myFlix REST API

A Web Server that provides information on movies. Built from the ground up on top of Node.js, with Express and MongoDB. The Authentication & Authorization are implemented using JWT tokens. Since myFlix is a public API it is used as backend for both: Topflix-React & TopFlix-Angular Applications. It is actually hosted on Heroku and the DataBase in Mongo Atlas.

See project on GitHub

Mobile Chat-App

A Chat Application for smartphones. Built with React Native, it uses Google's Firebase -with its Real Time Firestore Database- to store the chat history.

Pokedex App

A simple application that retrieves data from Pokemon characters through HTTP requests to a public API. Built with HTML, CSS, JavaScript, jQuery and Bootstrap.
