Some of my Work

Meet-App

This app gets a list of fictitious upcoming web development events around the world. It is a Progressive Web Application (PWA) written with a Test Driven Development (TDD) approach. The data comes from a private Google Calendar, fetched via the Google Calendar API by the means of serverless functions. It also implements an OAuth2 server, for users to login with their Google account credentials. The UI was built with React and includes chart visualizations that respond to changing events.

TopFlix Angular

A front-end Single Page Application (SPA) all about high-rated movies. Users of TopFlix can register/login to view all the films, display detailed information about their genre, director and synopsis, and select their favorites ones. Built with Angular, TypeScript and Angular Material. In conjunction with the myFlix web server, conforms a
MEAN fullstack application.

TopFlix React

A client-side Single Page Application (SPA) that showcases a list of recommended movies. Users can: register their own account to login; create their own list of favorite movies that persists in server, and view details about movie plots, genres and directors. Built with React, Redux and Bootstrap, TopFlix consumes the myFlix API conforming a
MERN fullstack application.

myFlix REST API

A Public Web Server that provides several endpoints to query information from handpicked movies. Built from the ground up with Node.js, Express and MongoDB, it implements authentication & authorization using JWT tokens. myFlix serves as the backend for both: Topflix-React & TopFlix-Angular applications.

Chat App

A mobile chat Application built with React Native and offline capabilities, that consumes Google's Firebase -with its Firestore Database- for real-time interactions. Users can chat, share pictures, photos and their current geolocation.

Pokedex

A front-end application that fetches Pokemon characters from a public API. Built with HTML, CSS, JavaScript, jQuery and styled using Bootstrap, users can search and display vector images and details from their favorite characters.
